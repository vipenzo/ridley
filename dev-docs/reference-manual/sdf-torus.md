# sdf-torus

## Signature

`(sdf-torus R r)`

## Description

Construct an SDF node for a torus in the XY plane, centered at the
origin with its axis along Z. `R` is the major radius (centre of the
tube to the torus axis); `r` is the minor radius (the tube's own
radius). Returns a lightweight SDF tree; no geometry is computed until
meshing.

`sdf-torus` is a convenience over `sdf-formula` plus annotated bounds —
internally it expands to a quoted radial distance expression with the
ring shape baked in. The auto-bounds carry the right axis-aligned box,
so meshing resolution scales correctly without manual hints.

> Desktop only: requires the libfive backend.

## Parameters

- `R` — major radius (distance from the centre of the tube to the
  torus axis).
- `r` — minor radius (tube radius). Must be less than `R` for a
  classic ring shape; `r ≥ R` produces a self-intersecting "spindle"
  torus.

## Example

```clojure
(register ring (sdf-torus 10 3))
```

A ring with a 3-unit tube circling at radius 10 around the Z axis.

## Notes

- To orient the torus along a different axis, use `rotate`
  (e.g. `(rotate ring :x 90)` puts it in the XZ plane).
- For a torus generated by revolving a custom 2D profile (any shape,
  not just a circle), use `sdf-revolve` with a 2D SDF profile.
